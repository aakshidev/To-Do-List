# 📅 Plan My Day

A simple and clean web-based productivity tool designed to help users organize their daily tasks, plan ahead, and manage different lists efficiently.

---

## ✨ Features

- 📝 **My Day**  
  Focus on tasks that need to be completed today.

- 📆 **Next 7 Days**  
  Plan upcoming tasks and stay ahead of deadlines.

- 📂 **Custom Lists**  
  Organize tasks into categories:
  - Personal  
  - Work  
  - Shopping  
  - Groceries  

- 🎯 **Minimal UI**  
  Clean and distraction-free design for better focus.

---

To run this project locally:

1. Clone the repository


2. Open the project folder  

3. Run the project  
Open `index.html` in your browser
